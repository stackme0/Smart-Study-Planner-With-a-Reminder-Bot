# Smart-Study-Planner-With-a-Reminder-Bot

## Description
A Java-based application that helps students plan their studies efficiently and provides emotionally aware reminders based on user mood.

## Features
- Task scheduling and management
- Deadline tracking
- Mood-based smart reminders
- File storage for tasks
